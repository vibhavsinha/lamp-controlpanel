lamp-controlpanel

This provides a frontend to using LAMP services. Allows the user to restart, start stop services from the LAMP stack. A good tool for web developers and designers who like to easily handle apache. Future versions will have ability to install the required web services softwares and php-modules from the front-end. It is also planned to have the ability to manage virtual hosts from the control-panel.
